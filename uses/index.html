Work Environment

  • VS Code: My primary IDE, used to open single files, scripting, markdown.
  • JetBrains: For large projects. PyCharm, Goland, Datagrip, etc.
  • Insomnia: REST client.
  • iTerm2: As Terminal, with tmux, zsh, oh-my-zsh, and powerlevel9k.
  • Alfred: This is awesome for daily desktop automation!
  • Homebrew: Which provides a lot of packages with fast & easy install.

Desktop & Web Apps

  • Tweetbot: Twitter client and clearly my favourite app! I love the fact my timeline is always synced between devices (Macbook, iPhone).
  • Shadow: For playing Windows games.
  • TT-RSS: My long-time RSS client since the death of Google Reader
  • Helium: A floating browser window. Useful when programming in fullscreen.
  • Textual: IRC client
  • Be Focused: Podomoro timer
  • MPV: Video player
  • Time Machine: Sync to NAS and Hubic Cloud
